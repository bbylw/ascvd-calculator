# PCE计算器

这是一个用于评估10年动脉粥样硬化性心血管疾病(ASCVD)风险的在线计算器。该计算器使用Pooled Cohort Equations方程进行风险评估。

## 功能
- 计算10年ASCVD风险
- 支持不同性别和种族的风险评估
- 响应式设计，支持移动端和桌面端
- 支持离线使用 (PWA)
- 支持单位转换 (mmol/L 与 mg/dL)
- 详细的风险评估报告和建议

## 在线使用
访问: [PCE计算器](你的cloudflare页面链接)

## 本地运行
1. 克隆仓库
2. 在浏览器中打开index.html文件

## 技术细节
- 使用标准PCE算法进行风险评估
- 支持基础PWA功能
- 无需后端，纯前端实现
- 符合2019 ACC/AHA指南

## 注意事项
- 适用年龄范围：40-79岁
- 不适用于既往有ASCVD病史的患者
- 计算结果仅供参考，具体诊疗请遵医嘱
